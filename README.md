# Vuejs based Poster Shop

Build This mini poster app using vuejs and imgur api.

#### Installation

1. Install this code on your local system

    ```    
    git clone https://github.com/sulemankhann/vuejs-poster-shop.git
    ```

2. Change directory into the local clone of the repository

    ```
    cd vuejs-poster-shop
    ```

3. Install dependencies

    ```
    npm install
    ```

4. Create a `.env` file by copying the sample

    ```
    cp .env_sample .env
    ```
    
    Or for Windows:
    
    ```
    copy .env_sample .env
    ```
    
    Now edit the *.env* file and replace the `IMGUR_CLIENT_ID` with the client ID provided in the pre-installation
    
5. Start project

    ```
    npm run start
    ```

6. Your site will be available at *localhost:[PORT]* where `PORT` is whatever value is set in your `.env` file.
